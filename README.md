# PDFUnlocker

一个用于解除PDF文件打印/复制/编辑权限限制的Python GUI工具。

**English Name:** PDFUnlocker

## 功能特点

- 📁 **拖拽上传** - 支持拖拽多个PDF文件到窗口进行批量解密
- 🖱️ **点击选择** - 点击空白区域可打开文件选择对话框
- 📊 **实时进度** - 显示解密进度条和处理状态
- 📁 **自定义导出** - 支持选择解密后文件的保存目录
- ✅ **批量处理** - 支持同时处理多个PDF文件
- ❌ **密码检测** - 自动检测并提示存在打开密码的文件

## 支持的解密类型

- 解除打印限制
- 解除复制限制
- 解除编辑限制
- 解除注释限制

> ⚠️ 注意：本工具仅能解除权限限制（如打印/复制/编辑），**无法破解PDF打开密码**。如果PDF文件设置了打开密码，工具会跳过该文件并提示用户。

## 安装依赖

### 方式一：使用 pip 安装

```bash
pip install PyMuPDF Pillow tkinterdnd2
```

### 方式二：使用 requirements.txt

```bash
pip install -r requirements.txt
```

## 使用方法

### 运行程序

```bash
python PDFUnlocker.py
```

### 使用步骤

1. **启动程序** - 运行 `PDFUnlocker.py`
2. **选择文件** - 拖拽PDF文件到窗口或点击窗口选择文件
3. **选择导出目录** - 默认保存到桌面，可点击"选择目录"更改
4. **等待完成** - 程序会自动解密并保存文件
5. **查看结果** - 处理完成后会弹出结果对话框

| 依赖包 | 版本 | 说明 |
|--------|------|------|
| PyMuPDF | >=1.23.0 | PDF处理核心库 |
| Pillow | >=9.0.0 | 图片处理库 |
| tkinterdnd2 | >=0.3.0 | 拖拽功能支持 |

## 技术实现

- **GUI框架**: Tkinter + tkinterdnd2
- **PDF处理**: PyMuPDF (fitz)
- **图片处理**: Pillow
- **平台支持**: Windows (原生窗口拖拽支持)

## 注意事项

1. 解密后的文件会在原文件名前加上"解密_"前缀
2. 请确保有足够的磁盘空间保存解密后的文件
3. 处理大文件时可能需要较长时间，请耐心等待
4. 如果遇到问题，请检查是否安装了所有依赖包
